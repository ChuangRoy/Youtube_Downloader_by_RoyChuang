# Youtube_Downloader

製作人：Roy Chuang

Email：s041516@apps.ntpc.edu.tw





## 特色

使用youtube-dl來做為下載影片的工具，Urllib.request內的urlopen用來確認網址是否正確 ，以及Python 內建的 tkinter GUI 來製作圖形使用者介面。





## 安裝

1. 使用pip 安裝工具安裝Urllib3

   `pip install urllib3`

2. 下載這裡的原始碼

3. 執行main.py

4. 可以開始使用了!!!
